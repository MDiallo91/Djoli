# School Management System

Application complète de gestion scolaire avec React, TypeScript et Electron.

## Fonctionnalités
- Enregistrement des élèves et gestion des dossiers
- Pointage et présence (Élèves & Enseignants)
- Structure de l'école (Classes, Matières, Années scolaires)
- Gestion des paiements et de la comptabilité

## Installation
```bash
npm install
npm run dev
```
